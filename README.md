# Algorithms
 Easy algorithms written as C# .NET library

## Usage
Add the /bin/Debug/netstandard2.0/Algorithms.dll as reference to your project.

## Changelog
* 2020-04-04 Initial commit with a few simple sorting algorithms

## Sorting algorithms
* Bubble Sort
* Heap Sort
* Insertion Sort
* Merge Sort
* Quick Sort

## Search algorithms
* Binary Search
